# GOVERNMENT FLOW BOT(DIALOGFLOW)

## Govflowbot
    -This is an chatbot developed in an **DIALOGFLOW** which is an google devloped environment to undestand and implement an natural language processing.
    -Dialogflow is an end-to-end, build-once deploy-everywhere development suite for creating conversational interfaces for websites, mobile applications, popular messaging platforms, and IoT devices.
    
## To setup :
    1.Login to Dialogflow and clone my repository
    2.Import the file in an dialog flow and test in an simulator or in an online simulator
